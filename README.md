# Know Thy Poop — Marketing site

Static marketing website for the [Know Thy Poop](https://app.pooptracker.site) app.

## Stack
Plain HTML + Tailwind (CDN) + Google Fonts. Zero build step.

## Files
- `index.html` — landing
- `terms.html` — Terms & Conditions
- `privacy.html` — Privacy Policy
- `icon.svg` — favicon / logo

## Local preview
Any static server works:
```
npx serve .
# or
python3 -m http.server 8080
```

## Deploy (Vercel)
This repo is deployed at [pooptracker.site](https://pooptracker.site).
No build command — Vercel serves static files directly.
